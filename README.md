# Semiparametric Heteroscedastic Gamma Regression for Sugarcane TSH Prediction

## Overview

This repository contains the datasets used in the study:

**"Semiparametric Heteroscedastic Gamma Regression for Sugarcane Tons of Sugar per Hectare (TSH) in India: A Distributional Modelling Framework Integrating Remote Sensing, Stress Indices, and Leaf Nutrient Status."**

The proposed SHGR model predicts both the mean and dispersion of Tons of Sugar per Hectare (TSH) using agronomic, meteorological, remote sensing, stress indices, and physiological variables collected from West Champaran district, Bihar, India.

## Dataset Description

The dataset integrates 19 data sources including:

1. Experimental plot registry
2. TSH records
3. Cultivar performance data
4. Landsat 8/9 OLI imagery
5. Cloud mask quality bands
6. Precipitation data
7. Temperature data
8. Soil bulk density
9. Pest pressure survey
10. Disease incidence survey
11. Leaf nitrogen content
12. Leaf phosphorus content
13. Leaf potassium content
14. GNDVI
15. Stress indices
16. Nutrient balance index
17. Chlorophyll and fluorescence parameters
18. Microclimate variables
19. Stomatal characteristics

---

## Response Variable

- Tons of Sugar per Hectare (TSH)

---

## Methodology

The proposed model is based on:

- Gamma distribution
- Generalized Additive Models for Location, Scale and Shape (GAMLSS)
- Penalized P-splines
- Semiparametric mean modelling
- Heteroscedastic dispersion modelling
- Landsat-derived GNDVI
- Abiotic and biotic stress indices

---

## Features

- Simultaneous prediction of TSH mean and variability
- Explicit modelling of heteroscedasticity
- Integration of remote sensing and physiological variables
- Quantification of drought, heat, pest and disease effects
- Risk-aware yield analysis
